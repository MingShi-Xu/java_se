# 基本数据类型

四类八种：

- 整型：byte  short  int  long

- 浮点型：float  double

- 字符型：string  char

- 布尔型：boolean

  #### 变量

  用来存储一块固定的内存区域，可以存储对应的数据类型的数据

  #### 命名规范

  可以使用英文、数字、_、$以及一部分unicode字符

  **注意：变量名必须以字母或下划线开头**

  #### 常量

  在数据类型前加final声明，一旦被赋值，就不能再次赋值。

  **注意：常量赋值只能进行一次**

  

  

# 数组

****定义格式：数据类型 数组名 = new 数据类型【数组长度】***

##### 数组初始化

在创建数组时对数组进行初始化赋值

**例：int[] arr = new int[] {1,2,3,4,5}; 简化  int arr = {1,2,3,4,5};**

##### 数组长度

数组一旦创建，长度就不变了

数组名.length调用数组长度

##### 数组初始化内容默认内容

基本数据类型：数值型默认为0(0.0)   字符型默认为null  布尔型默认为false
